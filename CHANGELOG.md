# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.0.0] - 2025-05-30
### Added
- **Mission Intelligence Center**: New dashboard page for real-time mission timeline and risk analysis.
- **Geographic Intelligence**: Global operations map with SVG flight routes and regional performance.
- **Predictive Analytics Dashboard**: Integrated ML models to predict component failures and estimate RUL.
- **Executive Insights Engine**: AI-driven insight cards generating plain-English operational observations.
- **Enterprise Portfolio Documentation**: Complete suite of recruiter-focused markdown assets, diagrams, and README.

### Changed
- Refactored frontend navigation to support 10 independent intelligence modules.
- Enhanced telemetry generators with injected seasonal noise and realistic equipment degradation curves.

## [2.0.0] - 2025-04-15
### Added
- Core Python ETL Pipeline linking raw synthetic data to the SQLite analytical database.
- 88 comprehensive DAX measures encompassing Time Intelligence and composite KPIs.
- Row-Level Security (RLS) implementation for 5 distinct business roles.

## [1.0.0] - 2025-03-01
### Added
- Initial proof-of-concept executive dashboard.
- Basic Star Schema design (Kimball methodology).
