## Capture Tips

1. **Resolution:** 1920×1080 or higher
2. **Browser:** Chrome/Edge with DevTools closed
3. **Zoom:** 100% (Ctrl+0 to reset)
4. **Wait:** Let all chart animations complete (~2 seconds after page load)
5. **Format:** PNG (lossless) for GitHub, JPEG for LinkedIn
6. **Crop:** Include the sidebar navigation to show page context
7. **Naming:** Use exactly the filenames above for README compatibility
