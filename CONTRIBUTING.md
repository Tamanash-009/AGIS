# AGIS — Contributing Guide

Thank you for your interest in contributing to the Anti-Gravity Intelligence System!

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/yourusername/AGIS.git`
3. **Create** a feature branch: `git checkout -b feature/your-feature-name`
4. **Commit** your changes: `git commit -m "feat: describe your change"`
5. **Push** to your fork: `git push origin feature/your-feature-name`
6. **Open** a Pull Request

## Commit Convention

We follow [Conventional Commits](https://www.conventionalcommits.org/):

- `feat:` — New feature
- `fix:` — Bug fix
- `docs:` — Documentation changes
- `style:` — Formatting, no code change
- `refactor:` — Code restructuring
- `test:` — Adding tests
- `chore:` — Build process, dependencies

## Code Standards

- Python: Follow PEP 8
- JavaScript: Use ES6+, `const`/`let` only
- SQL: Uppercase keywords, lowercase identifiers
- DAX: One measure per block, include comments

## Reporting Issues

Use GitHub Issues with the following labels:
- `bug` — Something isn't working
- `enhancement` — Feature request
- `documentation` — Documentation improvement
- `question` — General question
